# Box
You can upload your file and share it with everyone. 
This is an application using .Net7 and react.js using RTK Query.


**Backend**
- Framework:
-- ASP.NET Core 7.0 
---- Web API

- ORM:
-- EF Core 7.0.5

- Authentication:
-- Microsoft.AspNetCore.Authentication.JwtBearer 7.0.5
-- Microsoft.AspNetCore.OpenApi 7.0.5

- Test 
-- xUnit 2.4.2
-- Moq 4.x


**Frontend**
- Library
-- react 18.2.0
---- react-router-dom 5.x

- State Management
-- Reduxt-Toolkit
---- RTK-Query (normalized state structure)

- Tools
-- Vite
-- Axios
-- react-hook-form
-- react-icons
-- react-hot-toast

- Language
 -- TypeScript

- CSS Framework
 - Tailwind CSS
